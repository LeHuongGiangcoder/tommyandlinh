# Essential References

These are essential links referencing official documentation to master GSAP inside the context of your projects.

### Core Documentation
*   **[Official GSAP Cheatsheet](https://gsap.com/docs/v3/)**: Highly interactive, bookmark this.
*   **[Core Platform Documentation](https://gsap.com/docs/v3/GSAP)**: In-depth specifics of `.gsap()`, `.to()`, `.from()`, `.fromTo()`, timelines, defaults, and the configuration object.
*   **[Easing Visualizer](https://gsap.com/docs/v3/Eases)**: Essential for crafting unique animation "feels." Shows exactly how eases behave over time.

### React / Next.js Integration
*   **[@gsap/react Plugin Documentation](https://gsap.com/resources/React)**: Crucial documentation for proper integration within modern React functional components using the `useGSAP` hook.

### Key Plugins
*   **[ScrollTrigger Documentation](https://gsap.com/docs/v3/Plugins/ScrollTrigger/)**: Documentation for scroll-based animations: triggers, markers, pinning, and scrubbing.
*   **[SplitText Documentation](https://gsap.com/docs/v3/Plugins/SplitText/)**: (Premium plugin often useful for typograhpy animation) Guide on splitting strings into chars, words, and lines.
*   **[Flip Plugin Documentation](https://gsap.com/docs/v3/Plugins/Flip/)**: Powerful plugin for smooth layout transition state changes (simulating the FLIP technique).

### Learning Resources
*   **[GSAP "Getting Started" Hub](https://gsap.com/resources/get-started/)**: Comprehensive starting ground with videos.
*   **[GSAP Forums](https://gsap.com/community/)**: Extremely helpful, active community forum with extensive pre-existing codepens regarding complex visual problems.
